septembre 2010 - aujourd'hui :
Mission: Delta Dore, Combourg, France.
Projet 1 : développement de couches basses pour différentes cartes basées sur microcontrôleur PIC24F.

Contexte :
Delta Dore conçoit, fabrique et commercialise une offre complète de produits et services en domotique, gestion technique du bâtiment et maîtrise de l’efficacité énergétique. Dans le cadre du lancement d’une nouvelle gamme de produits s’appuyant sur le réseau LonWorks d’Echelon, mon rôle était de développer les couches basses pour les cartes électroniques de ce projet.
Ces couches basses ont été utilisées par la couche applicative de chaque carte.

  • Conception architecturale : Ecriture de l’architecture logicielle des couches basses (entrées/sorties binaires, gestion LED, entrées analogiques…)
  • Codage des couches basses : Codage en langage C et compilation sous MPLAB IDE Mise en place de briques pouvant être utilisées par plusieurs projets à l’aide de « customisation » Développement de versions de tests pour le client Communication UART
  • Tests unitaires

C bas niveau, makefile, serveur SVN, Doxygen, drivers, MPLAB, pic-gcc, UART, électronique analogique…

Projet 2 : développement sur carte contrôlant 4 ballasts lumière de type DALI.

Contexte :
Dans le monde du bâtiment d’entreprise, l’effort est de plus en plus marqué pour apporter des solutions d’économies d’énergie. Un des axes pour ces économiers est dirigé sur les luminaires. Le protocole DALI (Digital Addressable Lighting Interface) permet de contrôler intelligemment ces luminaires.

  • Développement d’une stack DALI pour gérer le protocole
  • Développement de l’applicatif sur la carte
  • Mise en place de tests de non-regression

C bas niveau, Lua, Cunit, makefile, serveur SVN, Cygwin, Doxygen, drivers, MPLAB, pic-gcc, UART, électronique analogique…
